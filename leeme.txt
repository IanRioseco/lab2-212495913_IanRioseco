Asegúrate de tener una carpeta con los siguientes archivos:
Archivos .pl (código Prolog): Incluyen todos los módulos y predicados necesarios para ejecutar el programa. Por ejemplo, tdagame.pl y otros relacionados.
Archivos .txt (scripts de prueba): Contienen los casos de prueba para verificar el funcionamiento del programa.
Archivo léeme.txt: Incluye información relevante sobre el uso de los archivos y detalles sobre su estructura.


Ve al menú superior de SWI-Prolog, selecciona File > Edit.
Navega hasta la carpeta donde están guardados los archivos .pl.
Selecciona un archivo y haz clic en Abrir. Repite este proceso para abrir todos los archivos .pl que forman parte del proyecto.


En el menú de SWI-Prolog, selecciona File > Consult.
Busca y selecciona el archivo tdagame.pl. Este archivo suele ser el punto de entrada del programa, ya que importa otros módulos y define las funcionalidades principales.
Al consultar este archivo, Prolog cargará todos los predicados definidos en los demás archivos vinculados.


Los archivos .txt contienen consultas prediseñadas para probar las funciones implementadas. Estas consultas están escritas en el formato de Prolog.
